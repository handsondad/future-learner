- **AI-Scientist**：开放式科学发现系统，全流程自动完成课题研究与论文写作。👉 [项目](https://github.com/PKU-DAIR/AI-Scientist)

- **AutoResearchClaw**：多 agent 协作的全流程科研系统。👉 [项目](https://github.com/handsome-rich/AutoResearchClaw)

- **Biomni**：聚焦生物医疗，自动推理与科研自动化。👉 [项目](https://github.com/stanford-biomni/Biomni)
- **DeepResearch**：文献检索&学术问答自动化。👉 [项目](https://github.com/SkyworkAI/DeepResearchAgent)
- **ScholarGPT**：AI学术检索推荐系统。👉 [项目](https://github.com/handsome-rich/ScholarGPT)
- **Claude Agent**：大模型协作的自动化实验。👉 [项目](https://github.com/anthropics/claude-agent)
